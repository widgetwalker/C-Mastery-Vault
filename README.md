## 🚀 Getting Started

To run any of these programs, you'll need a C compiler like `gcc` or `clang`.

1. **Clone the Vault**
   ```bash
   git clone https://github.com/USER_NAME/C-Mastery-Vault.git
   cd C-Mastery-Vault
   ```

2. **Compile and Execute**
   ```bash
   gcc 01-Fundamentals/hello.c -o hello
   ./hello
   ```
