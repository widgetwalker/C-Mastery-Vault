# ⚙️ System Programming

A look under the hood—simulating low-level system software components.

### 📜 Programs
- `assemblerSyntaxChecker.c` - Validates assembly opcodes and operands.
- `opcodeToObject.c` - Translates assembly-like instructions into object code.
- `addop.txt` - Sample input assembly code.
- `addobj.txt` - Generated object code mapping.
